# 人力招聘管理系统


> 可用于毕业设计参考或者个人学习用

- spring+srpingMVC+mybatic+mysql
- jsp

#### 实现功能
- [x] 公司企业负责人/面试者/的登录与登出  
- [x] 公司企业负责人发布岗位，管理岗位
- [x] 公司企业负责人查看邮件，查看简历，转发邮件，下载简历
- [x] 面试者可以搜索岗位，投递简历，管理自己的简历信息
